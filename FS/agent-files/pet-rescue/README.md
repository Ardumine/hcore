# 🐾 Paws & Claws Rescue Center — Website

A single-page HTML website for a fictional pet rescue center.

## Contents

- `index.html` — Full website with:
  - Header with branding
  - Navigation bar
  - Hero banner
  - **Adoptable pets** (6 featured animals: Buddy, Luna, Rocky, Mochi, Thumper, Peanut)
  - **About Us** section with location & hours
  - **Contact & Donate** section
  - **Wishlist** of needed supplies
  - Footer

## How to View

Open `index.html` in any web browser. All images are loaded from Unsplash.

## Tech

Pure HTML + CSS (no frameworks, no JavaScript). Responsive grid layout.
